Help bindings
 C-h
   ? : lets you see all help options
   ---
   a : apropos - look for things whose name matches a pattern
   ---
   k : describe-key
   c : describe-key-breifly
   b : describe-bindings
     It's hard to find c-x bindings, see:
     http://ergoemacs.org/emacs/gnu_emacs_keybinding_C-x.txt
   ---
   v : describe-variable
   f : describe-function
   o : describe-symbol
   P : describe-package
   m : describe-mode
   ---

 C-x
  C-f : find-file
  C-b : select a buffer
  C-e : eval-last-sexp
  C-c : save-buffers-and-kill-terminal (quit)
  k   : kill-buffer
